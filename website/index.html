CogChar is an open source cognitive character technology platform.

Our software is built using:

  • Java, Scala, and OSGi core technology
  • Basic symbolic smarts from Appdapter semantic computing platform
  • 3D character rendering to OpenGL using JMonkey Game Engine
  • Integration with MIDI music technology for action mapping, triggering, and synchronization
  • Serialization to Avro, integration with Hadoop
  • Connection with many other popular open source mathematics, AI and robotics tools

For more information, please visit the wiki in our workspace.
