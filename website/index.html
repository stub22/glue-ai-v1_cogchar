CogChar is an open source cognitive character technology platform.

Our project is built using:

  • Java, Scala, and OSGi core technology
  • Appdapter semantic computing platform
  • Integration with MIDI technology for action mapping and triggering
  • Serialization to Avro, Integration with Hadoop
  • Connection with many other popular AI and robotics tools

For more information, please visit our Wiki.
