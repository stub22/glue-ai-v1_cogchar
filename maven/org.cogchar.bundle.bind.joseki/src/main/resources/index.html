GLUE-AI.Cogchar HTTP-SPARQL Query/Update Server

 

Query forms

  • GLUE-AI SPARQL Query
  • GLUE-AI SPARQL Update
  • Query Joseski "books" file - good sanity check if other stuff isn't working.

Validators

  • SPARQL query validator
  • SPARQL update validator
  • RDF data validator

SPARQL Services

The forms above use the following services, which are bound in the WEB-INF/web.xml and joseki-config-friendu.ttl files of this bundle.

  • sparql/glue-ai
  • sparql-update/glue-ai
  • books : SPARQL query processor for a small book model in Data/books.ttl.
